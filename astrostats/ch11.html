Time Series Analysis

  • Time series analysis focuses on data collected over time, where observations could be correlated with past values. It aims to describe temporal patterns, make forecasts, and understand what is happening in time.
  • Basic components of a time series:
    • Trend – long-term increase or decrease.
    • Seasonality – systematic patterns repeating over fixed periods.
    • Cyclic behavior – longer-term oscillations without fixed frequency.
    • Noise – random variation not explained by structure.
  • Autocorrelation and partial autocorrelation:
    • The autocorrelation function (ACF) measures correlation between \(X_t\) and \(X_{t-k}\).
    • The partial autocorrelation function (PACF) removes intermediate lags to isolate direct dependence.
    • ACF and PACF plots help identify model structure.
  • AR, MA, and ARMA models describe linear dependence between a time series and its own past values.
    • Autoregressive (AR\((p)\)) model: \[ X_t = \phi_1 X_{t-1} + \cdots + \phi_p X_{t-p} + \varepsilon_t. \]
    • Moving Average (MA\((q)\)) model: \[ X_t = \varepsilon_t + \theta_1 \varepsilon_{t-1} + \cdots + \theta_q \varepsilon_{t-q}. \]
    • ARMA\((p,q)\) combines past observations and past errors.
  • ARIMA models incorporate differencing to make non-stationary data stationary:
    • ARIMA\((p,d,q)\) model:
      • \(d\) = number of differences to remove trend.
      • Useful for general forecasting of non-stationary series.
  • Exponential smoothing methods offer flexible forecasting approaches. Simple exponential smoothing (SES) is used for stationary data with no trend.
  • Frequency-domain analysis:
    • The spectral density function describes how variance is distributed across frequencies.
    • Useful for identifying periodic behavior.
    • The periodogram provides an empirical estimate of dominant frequencies.
  • Common accuracy metrics include MSE, RMSE, MAE, MAPE, and cross-validation on rolling windows. Residual diagnostics check whether any remaining structure in the data is random.
