Data smoothing: density estimation

  • Note about histograms: The effectiveness of a histogram can depend a lot on bin width. For this reason, it becomes subjective to use them and the intentions behind displaying a histogram a certain way should be fully explained in the text!
  • Kernel Density Estimation (KDE) is a nonparametric method for estimating a probability density function using a smooth kernel centered at each data point.
  • The KDE formula is: \[ \hat{f}(x,h)=\frac{1}{n h}\sum_{i=1}^{n} K\!\left(\frac{x-x_i}{h}\right) \] where \(K\) is the kernel function and \(h\) is the bandwidth (smoothing parameter).
  • Common kernels include: Gaussian \(K(y)=\frac{1}{\sqrt{2\pi}}e^{-y^2/2}\), and Uniform \(K(y)=\frac{1}{2}\) for \(|y|\le1\).
  • Bandwidth selection determines smoothness and is more important than the kernel choice. Silverman’s rule of thumb: \[ h=0.9\cdot \min\left(\mathrm{std}(x),\frac{\mathrm{IQR}(x)}{1.34}\right)n^{-1/5} \]
  • Adaptive Kernel Estimation uses a variable bandwidth, shrinking \(h\) in dense regions and enlarging it in sparse regions for improved accuracy.
  • Adaptive methods reduce oversmoothing/undersmoothing and better capture patterns when data density varies.
  • Nadaraya–Watson Estimator is a kernel regression method that estimates the conditional mean: \[ \hat{r}(x)= \frac{\sum_{i=1}^n y_i K\!\left(\frac{x-x_i}{h_x}\right)} {\sum_{i=1}^n K\!\left(\frac{x-x_i}{h_x}\right)} \] It behaves like a locally weighted average.
  • Spline smoothing fits piecewise polynomials joined smoothly at knots, producing a flexible nonparametric regression curve without assuming a specific functional form.
  • LOESS (Locally Estimated Scatterplot Smoothing) is a flexible nonparametric regression method that fits simple models (usually local weighted polynomials) to small neighborhoods of the data. It uses a smoothing parameter (the span) that determines how much of the data is used in each local fit. Smaller spans capture more detail; larger spans produce smoother curves. The method adapts to local structure, handles nonlinear relationships well, and does not assume a global functional form.
